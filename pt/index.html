Meta-Acervos é uma interface para pesquisa simultânea em várias coleções de arte por características visuais, desenvolvida pela equipe do Grupo Acervos Digitais e Pesquisa, projeto temático financiado pela Fundação de Amparo à Pesquisa do Estado de São Paulo (Fapesp).

Conheça os detalhes do projeto em nosso site e acesse o código da plataforma no nosso GitHub.

Museus

Selecione um ou mais museus para ver as obras.
Use os botões [selecionar todos] e [limpar seleção] para desfazer a ação.

Técnica

Use os botões para mostrar apenas as obras que contenham as técnicas selecionadas.
Caso nenhuma esteja selecionada, serão mostradas todas as obras.

Datas

Use os sliders para limitar o período de obras a ser mostrado.

Fauna e Flora

Use os botões para mostrar apenas as obras que contenham os objetos selecionados.
