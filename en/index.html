Developed by the Digital Collections and Research Thematic Project, funded by FAPESP.

canvas: intro: To Use: >- Choose one or more collections to start. Use the filters to refine the searches and the visualizations menu to create different arrangements from the filtered works. Filters: >- The filters are used to select which works will be displayed. They change the number of works included in the visualizations. Visualizations: >- The visualizations present different ways to visually explore the works selected through the filters. Visualizations do not change the number of displayed works, only the order and positioning in which the works are arranged on the screen. AI Curations: >- Selections made automatically by artificial intelligence, based on visual and semantic features of the artworks. Export Images: >- Mosaics are images generated from the chosen filters and visualizations. Unlike the visualizations on screen, these images are not interactive, and their size varies according to the number of works included in the search. In the compositions, cropped objects are arranged according to their relative position and location in the original work. mobile: >- This website was developed for desktop viewing. --- {% include head.html %} {% include canvas.html lang="en" %} {% include overlays/results-overlay.html %} {% include overlays/detail-overlay.html %} {% include overlays/about-overlay.html %} {% include en/menus/filter-menu.html %}
